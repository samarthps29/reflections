# Reflections App

The app is live at - https://reflections.cyclic.app/

#### Shortcut Commands

| Shortcut           | Description       |
| ------------------ | ----------------- |
| ctrl + k           | Quick Search      |
| ctrl + s           | Save Note         |
| ctrl + shift + c   | Clear the editor  |
| ctrl + shift + z   | Enter Zen Mode    |
| ctrl + shift + z/b | Exit Zen Mode     |
| ctrl + shift + h   | Show/Hide Sidebar |
